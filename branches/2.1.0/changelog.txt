version 2.0.2 (released 2010-02-17)
-add: 
				
					Added 'root-xpath' parameter to StandardRequestHandler and StandardResponseHandler
					Added 'remove-namespaces' parameter to StandardRequestHandler and StandardResponseHandler.
                      This is used to remove namespace declarations from request and response XML. If
                      value * is given, then all namespace declarations are removed:
						
                      	<remove-namespaces:>
                      	   <namespace-uri>*</namespace-uri>
                      	</remove-namespaces:>
						

                      or

						
                      	<remove-namespaces:>
                      	   <namespace-uri>uri:namespace1</namespace-uri>
                      	   <namespace-uri>uri:namespace2</namespace-uri>
                      	</remove-namespaces:>
