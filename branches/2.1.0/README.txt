====================================
Welcome to the Cordys HTTP Connector
====================================
The Cordys HTTP Connector provides connectivity to web servers by POSTing XML data from the connector 
SOAP request method body. Response XML data returned by the web server is returned in the SOAP 
response method body. It used for REST style connectivity as well.
